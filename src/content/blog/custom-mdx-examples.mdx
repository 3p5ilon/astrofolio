## Why MDX works well for blogs

MDX gives you Markdown readability with component-level flexibility.

### Inline emphasis

Use **bold text** for highlights and _italics_ for subtle emphasis.

### Code blocks

```ts
const isReadable = true;
const style = "minimal";
```

### Links

Use links to point readers to references, like the [Astro docs](https://docs.astro.build/).

### Math (KaTeX)

Inline math works like this: $E = mc^2$ and $a^2 + b^2 = c^2$.

Block math:

$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$

And a matrix example:

$$
\begin{bmatrix}
1 & 2 \\
3 & 4
\end{bmatrix}
$$
